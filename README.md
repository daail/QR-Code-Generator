# QR-Code-Generator
A lightweight, responsive web application that allows users to generate high-quality QR codes instantly from text or URLs. This project demonstrates clean DOM manipulation and integration with client-side logic.

🚀 Features

Real-time Generation: Convert text or URLs into QR codes instantly.

Downloadable Assets: Option to save the generated QR code as an image file.

Responsive Design: Fully optimized for desktop, tablet, and mobile viewing.

Clean UI: Minimalist interface focused on user experience.

🛠️ Tech Stack

Frontend: HTML5, CSS3

Scripting: JavaScript (ES6+)
